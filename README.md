## Run Locally

 

Clone the project

 

```bash

  git clone https://github.com/0xbm/Vehicles

```

 

Go to the project directory

 

```bash

  cd Vehicles

```

 

Install dependencies

 

```bash

  pip install -r requirements.txt

```

 

Start the server

 

```bash

127.0.0.1:8000

```
